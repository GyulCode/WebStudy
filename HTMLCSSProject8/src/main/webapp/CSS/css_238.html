더불어민주당 2021년 전당대회 돈봉투 의혹의 최종 수혜자로 지목된 송영길 전 대표가 7일 다시 한 번 자진 출두했으나 검찰이 거부했다. 송 전 대표는 이날 오전 9시 23분쯤 서울중앙지검에 도착해 수사팀에 면담을 요청했지만 거절당했다. 조사 일정이 잡히지 않아 출석이 거부된 것으로, 송 전 대표는 청사 로비에서 발길을 돌렸다. 그는 지난달 2일에도 자진 출두했으나, 검찰은 '조율되지 않은 일정'이라며 로비에서 돌려보냈다. 당시 송 전 대표는 1층 현관에서 기자회견을 열고 "귀국한 지 일주일이 지났지만, 검찰은 저를 소환하지 않고 주변 사람을 괴롭히고 있다"며 "검찰은 주위 사람을 괴롭히지 말고 저 송영길을 구속해주시기를 바란다"고 말했다.
